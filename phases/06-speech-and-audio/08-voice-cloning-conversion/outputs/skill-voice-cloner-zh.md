给定任务（语言、可用参考音频长度、适配预算、许可证约束、同意状态、部署规模），输出：

1. 方式。零样本克隆（F5-TTS / VibeVoice / Orpheus / OpenVoice V2）· 音色转换（kNN-VC / OpenVoice V2 音色迁移）· 说话人适配（XTTS v2 + LoRA / VITS 全量微调）。
2. 参考音频准备。所需长度、信噪比（SNR ≥ 20 dB）、单声道 16 kHz+、静音修剪、`ref_text`（F5-TTS 要求精确匹配）。拒绝含音乐背景的参考音频。
3. 同意证明。音色拥有者的明确录音同意。模板：姓名 + 日期 + 目的 + 范围 + 撤回程序。保存 7 年以上。
4. 水印。每个输出上嵌入 AudioSeal 16 位载荷。在 CI 中配置检测器，在发布音频前验证水印存在。
5. 安全过滤器。命名实体（名人 / 政客 / 未成年人）提示拒绝；每用户每小时速率限制；每次克隆生成的审计日志；紧急关闭开关。

拒绝在没有水印策略的情况下交付克隆。拒绝克隆知名名人/政客/未成年人，无论其声称的同意如何。拒绝时长低于 3 秒或 SNR < 20 dB 的参考音频。拒绝将 F5-TTS 用于商业部署（CC-BY-NC）。拒绝在没有明确标记口音迁移差距的情况下进行跨语言克隆。
