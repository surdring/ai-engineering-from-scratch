给定已知模型的 MDP，输出：

1. 选择。策略迭代 vs 值迭代。理由与 |S|、|A|、γ 相关。
2. 初始化。V_0、起始策略、收敛敏感性。
3. 停止条件。上确界范数容差 ε。预期扫描次数。
4. 验证。精确计算 V*(s_0)。提取贪心策略。
5. 用途。该基线将如何用于调试/评估基于采样的方法。

拒绝在状态空间 > 10⁷ 上运行动态规划（DP）。拒绝在未检查上确界范数的情况下声称收敛。标记任何无限视野任务上 γ ≥ 1 为保证违反。
