## 学习目标

- 从零计算熵、交叉熵和 KL 散度，并解释它们的关系
- 推导为什么最小化交叉熵损失等价于最大化对数似然
- 计算特征与目标之间的互信息以排序特征重要性
- 将困惑度解释为语言模型选择的有效词汇量大小

## 问题

你在每个分类模型中都调用 `CrossEntropyLoss()`。你在每篇语言模型论文中都看到「困惑度」。你在 VAE、蒸馏和 RLHF 中读到 KL 散度。这些不是不相关的概念。它们都是同一个想法戴着不同的帽子。

信息论给你推理不确定性、压缩和预测的语言。克劳德·香农在 1948 年发明了它来解决通信问题。事实证明，训练神经网络就是一个通信问题：模型试图通过学习权重的噪声信道传输正确的标签。

## 概念

### 信息量（惊讶）

当不太可能的事情发生时，它携带更多信息：

```
I(x) = -log(p(x))
```

使用以 2 为底的对数得到比特。使用自然对数得到 nat。同样的思想，不同的单位。

确定事件携带零信息。你已经知道它们会发生。

### 熵（平均惊讶）

熵是分布所有可能结果中的期望惊讶：

```
H(P) = -sum( p(x) * log(p(x)) )  对所有 x
```

公平硬币对于二元变量具有最大熵：1 比特。有偏硬币（99% 正面）具有低熵：0.08 比特。熵衡量分布中不可约简的不确定性。你无法将其压缩到熵以下。

### 交叉熵（你每天使用的损失函数）

交叉熵衡量当你使用分布 Q 来编码实际来自分布 P 的事件时的平均惊讶：

```
H(P, Q) = -sum( p(x) * log(q(x)) )  对所有 x
```

P 是真实分布（标签）。Q 是你模型的预测。如果 Q 完美匹配 P，交叉熵等于熵。任何不匹配都会使其变大。

在分类中，P 是 one-hot 向量，这简化交叉熵为：H(P, Q) = -log(q(真实类别))。这就是分类中交叉熵损失公式的全部。最大化正确类别的预测概率。

### KL 散度（分布之间的距离）

KL 散度衡量使用 Q 而不是 P 带来的额外惊讶：

```
D_KL(P || Q) = H(P, Q) - H(P)
```

交叉熵是熵加上 KL 散度。由于训练期间真实分布的熵是常数，最小化交叉熵等同于最小化 KL 散度。你在推动模型的分布接近真实分布。

KL 散度不对称：D_KL(P || Q) != D_KL(Q || P)。它不是真正的距离度量。

### 互信息

互信息衡量知道一个变量能告诉你关于另一个变量的多少信息：

```
I(X; Y) = H(X) - H(X|Y) = H(X) + H(Y) - H(X, Y)
```

如果 X 和 Y 独立，互信息为零。如果完全相关，互信息等于任一变量的熵。在特征选择中，特征和目标之间的高互信息意味着该特征有用。低互信息意味着它是噪声。

I(X;Y) 总是非负的，对称为 I(X;Y) = I(Y;X)。与 KL 散度不同，它是对称的。

### 条件熵

H(Y|X) 衡量在观察 X 后对 Y 还有多少不确定性。如果 X 完全决定 Y，则 H(Y|X) = 0。如果 X 对 Y 没有任何信息，则 H(Y|X) = H(Y)。条件熵永远非负且不超过 H(Y)：0 <= H(Y|X) <= H(Y)。

### 联合熵与信息图

H(X,Y) 是 X 和 Y 联合分布的熵。关键关系：
- H(X,Y) = H(X) + H(Y|X) = H(Y) + H(X|Y)
- I(X;Y) = H(X) - H(X|Y) = H(Y) - H(Y|X)
- H(X,Y) = H(X) + H(Y) - I(X;Y)
- H(X,Y) <= H(X) + H(Y)，等号当 X 和 Y 独立时成立

### 困惑度

在语言模型中，困惑度是每个词的交叉熵的指数：perplexity = exp(cross_entropy)。直观解释：困惑度是你模型在每一步从中选择的有效词汇量大小。均匀分布（所有 50,000 个词等可能）的困惑度是 50,000。完美模型的困惑度是 1——它确切知道下一个词是什么。困惑度越低意味着模型越好。

## 关键术语

| 术语 | 人们常说的 | 实际含义 |
|------|-----------|---------|
| 熵 | "不确定性" | 来自分布的期望惊讶 |
| 交叉熵 | "损失函数" | 使用错误分布产生的平均惊讶 |
| KL 散度 | "距离" | 使用一个分布代替另一个的额外成本 |
| 互信息 | "共享信息" | 知道一个变量能减少的另一个变量的不确定性 |
| 困惑度 | "分支因子" | 语言模型在每步需要选择的等效词数 |
