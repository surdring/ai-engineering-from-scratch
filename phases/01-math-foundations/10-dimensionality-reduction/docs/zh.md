# 降维

> 高维数据有结构。你通过从正确的角度观察来发现它。

**类型：** 构建
**使用语言：** Python
**前置课程：** 阶段 1，第 01-03、06 课
**预计时间：** ~90 分钟

## 学习目标

- 从零实现 PCA：中心化数据、计算协方差矩阵、特征分解、投影
- 使用解释方差比和肘部法选择主成分数量
- 比较 PCA、t-SNE 和 UMAP 在二维中可视化 MNIST 数字的效果
- 应用带 RBF 核的核 PCA 来分离标准 PCA 无法处理的非线性数据结构

## 问题

每个样本有 784 个特征。也许是手写数字的像素值。也许是基因表达水平。也许是用户行为信号。你无法可视化 784 个维度。你无法绘制它们。你甚至无法思考它们。

但这些 784 个特征大多冗余。实际信息存在于一个更小的曲面上。降维找到那个更小的曲面，将 784 维数据压缩到 2、10 或 50 维，同时保留重要的结构。

## 概念

### 维度灾难

高维空间不直观。随着维度增长，三个东西会崩溃。

**距离变得无意义。** 在高维中，任意两个随机点之间的距离收敛到相同值。最近邻搜索失效。

```
维度    平均距离比（随机点之间的最大/最小）
2      ~5.0
10     ~1.8
100    ~1.2
1000   ~1.02
```

**体积集中在角落。** d 维的单位超立方体有 2^d 个角。在 100 维中，几乎所有体积都在角落，远离中心。数据点分散到边缘，模型在内部缺乏数据。

**需要指数级更多的数据。** 要从 2D 到 20D 保持相同的数据密度，需要 10^18 倍的数据。你永远不会足够。

### PCA：找到重要的方向

主成分分析 (PCA) 找到数据变化最大的轴。它旋转你的坐标系，使第一个轴捕获最大方差，第二个捕获次大，依此类推。

算法：
1. 中心化数据（减去每列均值）
2. 计算协方差矩阵（特征如何共同变化）
3. 特征分解（找到主方向）
4. 按特征值排序（最大方差优先）
5. 投影（保留前 k 个特征向量，丢弃其余）

**解释方差比**告诉你有多少被捕获。当累积解释方差达到 0.95，你就知道这些组件捕获了 95% 的信息。

### 选择组件数量

三种策略：阈值（保留 90-95% 方差）；肘部法（绘制每个组件的解释方差，找急剧下降点）；下游性能（使用 PCA 作为预处理，扫描 k 并衡量模型准确率）。

### t-SNE：保留邻域

t-SNE（t-分布随机邻域嵌入）为可视化而设计。在原始空间中基于距离计算点对上的概率分布，然后找到这些分布保持的二维排列。784 维中的邻居在 2D 中仍是邻居。

关键特性：非线性、随机（不同运行产生不同布局）、困惑度参数控制邻居数量（5-50）、簇间距离无意义、大数据集上慢（O(n^2)）。

### UMAP：更快，更好的全局结构

UMAP（均匀流形近似与投影）类似 t-SNE 但更快且全局结构更好。它使用近似最近邻图。

关键参数：n_neighbors（多少邻居定义局部结构，较高值保留更多全局结构）；min_dist（输出中点打包的紧密程度）。

### 何时用哪种方法

| 方法 | 用例 | 保留 | 速度 |
|------|------|------|------|
| PCA | 训练前预处理 | 全局方差 | 快 |
| PCA | 快速探索性可视化 | 线性结构 | 快 |
| t-SNE | 出版质量二维图 | 局部邻域 | 慢（<10k 样本） |
| UMAP | 大规模二维可视化 | 局部+部分全局结构 | 中 |
| t-SNE/UMAP | 理解聚类结构 | 聚类分离 | 中到慢 |

### 核 PCA

标准 PCA 找线性子空间。如果数据在非线性流形上呢？核 PCA 通过核函数在诱导的高维特征空间中应用 PCA，无需显式计算坐标。这是核技巧——与 SVM 背后的思想相同。

常见核函数：RBF（高斯）核 exp(-gamma * ||x-y||^2) 适用于大多数非线性数据；多项式核 (x·y + c)^d。

### 重构误差

重构误差 = 未包含的特征值之和。总方差 = 所有特征值之和。丢失比例 = （丢弃的特征值之和）/（所有特征值之和）。重构误差还可用于异常检测：高重构误差的样本是不符合学习子空间的异常值。

## 关键术语

| 术语 | 人们常说的 | 实际含义 |
|------|-----------|---------|
| PCA | "旋转+丢弃" | 找到并保留方差最大的方向 |
| 特征值 | "方差量" | 沿对应特征向量方向有多少方差 |
| 解释方差比 | "信息保留" | 单个组件捕获的总方差比例 |
| t-SNE | "邻域绘图" | 在 2D 中保持哪些点靠近的非线性嵌入 |
| UMAP | "快速 t-SNE" | 带更好全局结构和更快的 t-SNE 后继者 |
| 核 PCA | "非线性 PCA" | 通过核函数在隐式高维空间中操作的 PCA |
