# 凸性检查器（Convexity Checker）

如何验证一个优化问题是否为凸的，以及得到答案后该怎么做。

## 决策清单

1. 目标函数是否凸？（检查 Hessian 矩阵半正定性或使用复合规则。）
2. 所有不等式约束是否形式为 g_i(x) <= 0，其中每个 g_i 是凸的？
3. 所有等式约束是否都是仿射（线性）的？
4. 如果以上三项都是，该问题是凸的。使用具有收敛保证的凸求解器。
5. 如果任何一项不成立，该问题是非凸的。使用 SGD/Adam 并接受局部最优解。

## 如何测试函数的凸性

| 测试方法 | 适用范围 | 方法 |
|---|---|---|
| 二阶导数 >= 0 | 标量函数 f(x) | 计算 f''(x)。如果对所有 x 有 f''(x) >= 0，则凸。 |
| Hessian 半正定 | 多元函数 f(x) | 计算 H(x)。如果所有特征值处处 >= 0，则凸。 |
| 定义检验 | 任意函数 | 对采样的 x, y, t，检查 f(tx + (1-t)y) <= t*f(x) + (1-t)*f(y)。 |
| 复合规则 | 复合函数 | 参见下方的复合规则表。 |
| 限制到一条直线 | 多元函数 f | f 是凸的当且仅当 g(t) = f(x + tv) 对所有 x, v 关于 t 是凸的。 |

## 复合规则（保持凸性）

| 操作 | 结果 |
|---|---|
| f + g（两者均凸） | 凸 |
| c * f（c > 0，f 凸） | 凸 |
| max(f, g)（两者均凸） | 凸 |
| f(Ax + b)，其中 f 凸 | 凸 |
| g(f(x))，其中 g 凸非减且 f 凸 | 凸 |
| g(f(x))，其中 g 凸非增且 f 凹 | 凸 |
| 凸函数之和 | 凸 |
| 凸函数的逐点上确界 | 凸 |

## 常见机器学习目标函数：凸还是非凸？

| 目标函数 | 凸？ | 原因 |
|---|---|---|
| MSE：(1/n) sum(y - Xw)^2 | 是 | 关于 w 的二次型，Hessian = (2/n) X^T X 半正定 |
| Logistic 损失：sum(log(1 + exp(-y_i * w^T x_i))) | 是 | 凸函数之和（log-sum-exp 族） |
| Hinge 损失：sum(max(0, 1 - y_i * w^T x_i)) | 是 | 凸（线性）函数的最大值 |
| L2 正则化：lambda * \|\|w\|\|^2 | 是 | 二次型，Hessian = 2*lambda*I |
| L1 正则化：lambda * \|\|w\|\|_1 | 是 | 绝对值之和（凸但不可微） |
| 岭回归（Ridge Regression）：MSE + L2 | 是 | 两个凸函数之和 |
| LASSO：MSE + L1 | 是 | 两个凸函数之和 |
| Elastic Net：MSE + L1 + L2 | 是 | 凸函数之和 |
| SVM（原始形式）：hinge + L2 | 是 | 凸函数之和 |
| 带 softmax 的交叉熵 | 是（在对数空间） | Log-sum-exp 是凸的 |
| 神经网络（任意损失） | 否 | 非线性激活产生非凸复合 |
| k-means 目标函数 | 否 | 离散分配步骤 |
| 矩阵分解：\|\|X - UV^T\|\|^2 | 否 | 关于 U 和 V 是双线性的 |
| GAN 损失 | 否 | 极小极大问题，关于生成器非凸 |
| 对比损失（InfoNCE） | 否 | 指数与负样本的对数比值 |

## 基于凸性的求解器选择

| 问题类型 | 求解器 | 收敛保证 |
|---|---|---|
| 凸、光滑、无约束 | 梯度下降 | O(1/k) 到全局最小值 |
| 凸、光滑、无约束 | L-BFGS | 超线性收敛到全局最小值 |
| 凸、光滑、无约束 | 牛顿法 | 在最小值附近二次收敛（如果 Hessian 可计算） |
| 凸、光滑、有约束 | 内点法 | 多项式时间 |
| 凸、非光滑（L1） | 近端梯度 / ISTA | O(1/k) 到全局最小值 |
| 凸、非光滑（L1） | ADMM | 灵活，可处理约束 |
| 凸、二次型 | 共轭梯度 | n 步内精确 |
| 非凸、光滑 | SGD / Adam | 收敛到局部最小值 |
| 非凸、光滑 | SGD + 重启 | 平均意义上更好的局部最小值 |
| 非凸、光滑 | 过参数化 + SGD | 平坦极小值，良好泛化 |

## 常见错误

- 因为损失函数凸就假设问题凸。损失必须在所优化的参数上是凸的。交叉熵在对数空间中关于参数是凸的，但从输入到对数的完整神经网络映射是非凸的。
- 对非凸问题使用牛顿法。Hessian 可能有负特征值，导致牛顿法向鞍点或最大值移动而非最小值。
- 忘记 L1 正则化使目标函数在零点不可微。标准梯度下降效果不佳。使用近端梯度下降或次梯度方法。
- 通过构造 A^T A 对条件数求平方。如果需要求解最小二乘问题且 A 病态，使用 QR 或 SVD 而非正规方程（Normal Equation）。
- 未经检查就认定问题非凸。许多机器学习问题（线性模型、SVM、逻辑回归）是凸的，可以受益于更强的求解器。

## 快速测试：我的问题是否凸？

```
1. 写出目标函数：在约束条件下最小化 f(w)
2. 对 f(w) 中的每一项：
   - 是否为带有半正定矩阵的二次型？ -> 凸
   - 是否为范数？ -> 凸
   - 是否为 log-sum-exp？ -> 凸
   - 是否包含 w 的非线性（sigmoid(w), w1*w2）？ -> 可能非凸
3. 所有约束是否都是线性或凸不等式？
4. 如果所有项都凸且约束凸/线性 -> 问题凸
5. 如果任何一项非凸 -> 问题非凸
```
