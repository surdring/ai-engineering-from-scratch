# 凸优化

> 凸问题有一个山谷。神经网络有数百万个。知道区别很重要。

**类型：** 构建
**使用语言：** Python
**前置课程：** 阶段 1，第 04、08 课
**预计时间：** ~90 分钟

## 学习目标

- 使用定义、二阶导数和 Hessian 准则测试函数是否为凸函数
- 实现牛顿法并与梯度下降比较其二次收敛速度
- 使用拉格朗日乘子解约束优化问题，解释 KKT 条件
- 解释为什么神经网络损失景观是非凸的，但 SGD 仍能找到好的解

## 问题

第 08 课教了梯度下降、动量和 Adam。这些优化器在任何曲面上走下坡。但它们没有保证。非凸景观上的梯度下降可能落入糟糕的局部极小值，卡在鞍点上，或永远振荡。

但机器学习中许多问题是凸的。线性回归、逻辑回归、SVM、LASSO、岭回归。对这些，存在更强的工具：带有数学保证的优化。凸问题恰好有一个山谷。任何走下坡的算法都会到达全局极小值。不需要重启、学习率调度或祈祷。

## 概念

### 凸集

如果集合中任意两点之间的线段也完全在该集合中，则该集合是凸的。矩形、三角形是凸的。星形、月牙形、甜甜圈形不是凸的。

### 凸函数

如果对任意 x、y 和 t∈[0,1]：f(tx+(1-t)y) <= t*f(x)+(1-t)*f(y)。几何上：曲线上任意两点之间的线段位于曲线之上或之上。

常见凸函数：f(x)=x^2、f(x)=|x|、f(x)=e^x、f(x)=-log(x)（x>0）、任何线性函数。

### 测试凸性

测试 1（1D 二阶导数）：f''(x) >= 0 对所有 x → 凸。测试 2（多元 Hessian）：H(x) 正半定对所有 x → 凸。测试 3（定义）：直接检查不等式。

### 凸性为何重要

凸优化的核心定理：凸函数的每个局部极小值都是全局极小值。梯度下降不可能被困。任何下坡路径都通向同一答案。算法保证收敛到最优解。

### ML 中的凸 vs 非凸

| 问题 | 凸？ | 原因 |
|------|-----|------|
| 线性回归 (MSE) | 是 | 损失是权重的二次函数 |
| 逻辑回归 | 是 | 对数损失在权重上是凸的 |
| SVM | 是 | 线性函数的最大值 |
| LASSO（L1） | 是 | 凸函数之和是凸的 |
| Ridge（L2） | 是 | 二次+二次=凸 |
| 神经网络 | 否 | 非线性激活创造非凸景观 |
| k-means | 否 | 离散分配步骤 |

线性模型加凸损失是凸的。一旦添加带非线性激活的隐藏层，凸性就破坏了。

### Hessian 矩阵

H[i][j] = d^2f/(dx_i dx_j)。Hessian 描述曲率：所有特征值正（向上弯曲=凸点）、全负（向下弯曲=局部极大值）、混合符号（鞍点）、零特征值（退化）。

### 牛顿法

使用二阶信息（Hessian）而非仅一阶（梯度）。在当前点拟合二次近似，直接跳向该二次的最小值。x_new = x - H^(-1) * gradient。优点：最小值附近二次收敛、无需调节学习率、尺度不变。缺点：计算 Hessian O(n^2) 并求逆 O(n^3)——对百万权重的网络不实用。

### 约束优化与拉格朗日乘子

最小化 f(x) 受 g(x)=0 约束 → 构造拉格朗日函数 L(x,λ) = f(x) + λ*g(x)。解 ∇L = 0 求 x 和 λ。不等式约束扩展到 KKT 条件。SVM 的关键洞察：引入拉格朗日乘子将约束问题转化为对偶问题，揭示了核技巧。

### 对偶性

原问题：min f(x) s.t. g(x) <= 0。对偶函数：d(λ) = min_x L(x,λ) = min_x [f(x) + λ*g(x)]。弱对偶：d(λ*) <= f(x*)——对偶总是给出下界。强对偶（凸+满足约束资质时）：d(λ*) = f(x*)——对偶给出精确答案。SVM 使用对偶性引出了核技巧的优雅数学。

## 关键术语

| 术语 | 人们常说的 | 实际含义 |
|------|-----------|---------|
| 凸函数 | "一个山谷" | 每个局部极小值也是全局极小值 |
| 牛顿法 | "基于曲率的步骤" | 使用 Hessian（二阶信息）进行更快的二次收敛 |
| 拉格朗日乘子 | "约束平衡" | 将约束纳入目标函数以实现无约束优化 |
| KKT 条件 | "最优性检查" | 约束优化的最优必要条件 |
| 对偶性 | "从上方逼近" | 最大化下界问题以解原始问题 |
