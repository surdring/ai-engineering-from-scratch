# 机器学习中的统计学

> 统计学告诉你模型是真的有效还是只是运气好。

**类型：** 构建
**使用语言：** Python
**前置课程：** 阶段 1，第 06-07 课
**预计时间：** ~120 分钟

## 学习目标

- 从零计算描述性统计量、Pearson/Spearman 相关系数和协方差矩阵
- 进行假设检验（t 检验、卡方检验），正确解释 p 值和置信区间
- 使用自助法重采样为任意度量构造置信区间，无需分布假设
- 使用效应量度量区分统计显著性与实际显著性

## 问题

你训练了两个模型。A 得分 0.87。B 得分 0.89。你部署了 B。三周后生产指标比之前更差。发生了什么？

模型 B 并没有真正优于 A。0.02 的差异是噪声。测试集太小，或方差太高，或两者兼有。你发布了伪装成改进的随机性。

这不断发生。Kaggle 排行榜震动。无法复现的论文。基于几百个样本宣布赢家的 A/B 测试。根本原因总是一样：有人跳过了统计学。

## 概念

### 描述性统计：总结数据

**集中趋势的度量：**
- 均值：总和/计数（平衡点）
- 中位数：排序后的中间值（抗异常值）
- 众数：最频繁的值

当均值和中位数偏离时，分布是偏斜的。

**散布度量：**
- 方差：与均值的平均平方偏差
- 标准差：方差的平方根，与数据相同单位
- IQR：Q3-Q1（四分位距），中间 50% 数据的范围，抗异常值

**百分位数：** P50 是中位数，P95 是坏但不是最坏的情况，P99 是尾延迟（通常十倍于中位数）。在 ML 中用于推理延迟、预测置信度分布和误差分布。

**样本 vs 总体：** 从样本计算方差时除以 (n-1) 而非 n——贝塞尔校正。

### 相关性：变量如何一起变动

**Pearson 相关系数：** 衡量线性关联，从 -1 到 +1。假设线性关系且两个变量大致正态分布。对异常值敏感。

**Spearman 秩相关：** 衡量单调关联。用秩替换每个值再计算 Pearson。捕获任何单调关系，不仅仅是线性的。对异常值稳健。

黄金法则：相关性不意味着因果。冰淇淋销量和溺水死亡相关，因为两者都在夏天增加。

### 协方差矩阵

对于 d 个特征，协方差矩阵 C 是 d x d 矩阵，其中 C[i][j] = Cov(feature_i, feature_j)。对角线项是每个特征的方差。与 PCA 的联系：PCA 对协方差矩阵进行特征分解——特征向量是主成分，特征值告诉你每个成分捕获了多少方差。

### 假设检验

**设置：** H0（零假设）= 默认假设，通常是「无效果」；H1（备择假设）= 你试图证明的。

**p 值**是在 H0 为真的假设下，看到像你观察到的数据那样极端的概率。它不是 H0 为真的概率。如果 p < alpha（通常 0.05），拒绝 H0。如果 p >= alpha，未能拒绝 H0——这并不意味着 H0 为真。

**置信区间：** 95% CI = x_bar ± 1.96 * (s/sqrt(n))。解释：如果你重复此实验多次，95% 的计算区间将包含真实均值。不是「有 95% 概率真实均值在此区间内」。

### t 检验

比较两组的均值。独立样本 t 检验比较两个独立组。配对 t 检验比较相同的样本在处理前后。t 统计量衡量以标准误差为单位的差异，较大的 t 值意味着更强的证据。在 ML 中使用 t 检验比较模型在多个试验中的准确率。

### 自助法

从你的数据中有放回地重复采样，计算每次重采样的统计量，查看统计量分布。无需假设数据遵循正态分布。为任意统计量（不仅仅是均值）构造置信区间。对 ML 指标如准确率、F1 分数、AUC 特别有用。

### 统计显著性 vs 实际显著性

统计显著性（p < 0.05）不等于实际重要性。大样本量下，微小的差可能统计显著但在实践上无意义。效应量度量差异的大小：Cohen's d 衡量均值差除以标准差，d=0.2 小，d=0.5 中，d=0.8 大。始终同时报告 p 值和效应量。

### 多重比较

如果你测试 20 个假设，纯随机会得到约一个 p < 0.05 的结果。Bonferroni 校正：将 alpha 除以测试数量。这对模型比较很重要——如果你比较 10 个超参数组合，显著性阈值是 0.005 而非 0.05。

## 关键术语

| 术语 | 人们常说的 | 实际含义 |
|------|-----------|---------|
| p 值 | "显著性分数" | 假设零假设为真，看到如此极端数据的概率 |
| 置信区间 | "合理范围" | 基于数据对参数的合理值范围 |
| 相关性 | "关系强度" | 两个变量共同变动的程度 |
| 自助法 | "重采样" | 有放回地重复采样以估计变异性 |
| 效应量 | "实际重要性" | 差异的大小，而不仅仅是是否存在 |
