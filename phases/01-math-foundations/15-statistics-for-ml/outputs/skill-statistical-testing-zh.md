# ML 统计检验

如何为给定的实验设置选择合适的统计检验。

## 决策清单

1. 是比较两个模型还是一组模型？两个用成对检验，多个用 ANOVA。
2. 数据是成对的（相同的训练/测试划分）还是独立的？
3. 分布形状是已知的（正态性）还是未知的？正态用 t 检验，否则用非参数检验。
4. 度量的类型是什么？准确率（比率）与 MSE（连续）使用不同的检验。
5. 多重比较是否是个问题？使用 Bonferroni 校正或更强大的 FWER 控制方法。

## 检验选择框架

```
比较两个模型/条件？
  是 --> 数据成对？
       是 --> 差值正态 + N > 30？ --> 成对 t 检验
        |   差值非正态？ --> Wilcoxon 符号秩检验
        |   分类度量（如准确率）？ --> McNemar 检验
       否 --> 正态性 + 等方差？ --> 独立 t 检验
           |   等方差存疑？ --> Welch t 检验
           |   非正态？ --> Mann-Whitney U 检验
  否 --> 比较 3+ 个模型？
       是 --> 正态性 + 等方差 + 独立？ --> 单因素 ANOVA
        |   非正态或不等方差？ --> Kruskal-Wallis H 检验
        |   重复测量/交叉验证折叠？ --> Friedman 检验
```

## 常见 ML 比较 —— 使用哪个检验

| 场景 | 检验 | 备注 |
|------|------|------|
| 交叉验证中模型 A vs B（所有折叠相同） | 成对 t 检验 | 假设 CV 分数差正态分布 |
| A vs B，使用独立测试集 | 独立 t 检验 | 前提假设较少 |
| A vs B，准确率/错误率更优 | McNemar 检验 | 专门针对分类器差异 |
| 同一褶皱上 5 折 CV 中多个模型 | Friedman + Nemenyi | 通过 Nemenyi 进行组内比较 |
| 比较 10+ 个模型 | 弗里德曼（Friedman） | 避免 t 检验的多重比较陷阱 |
| 两个模型的多个指标（准确率、F1 等） | MANOVA 或逐个 + Bonferroni 校正 | 同时考虑所有指标 |
| 只有一个模型检查改进是否显著 | 单样本 t 检验（对比基线） | 与固定基线比较 |
| 非正态 AUC 分布 | DeLong 检验 | 专门为 ROC 比较设计 |

## 各检验的详细说明

### McNemar 检验（用于分类器比较）

仅适用于二分类准确率。检查两个模型之间的分歧在哪里。

```
             模型 B 正确    模型 B 错误
模型 A 正确      a             b
模型 A 错误      c             d

检验统计量：(|b - c| - 1)^2 / (b + c)  ~ 卡方分布，df=1
```

解释：模型 A 正确但 B 错误（b）与模型 B 正确但 A 错误（c）的样本数。如果 b >> c，则 A 显著更好。

### Friedman 检验（用于多个模型比较）

非参数检验，对所有模型在相同数据集上的性能进行排序。

```
1. 对每个数据集（或 CV fold），将模型按性能排名（最好的排名为 1）
2. 计算每个模型的平均排名
3. 检验统计量基于排名差异度量与零假设下期望的比值
4. 如果检验显著，至少有一个模型与其他模型不同
5. 使用 Nemenyi 进行事后两两比较
```

### Wilcoxon 符号秩检验

t 检验的非参数替代。用于比较成对样本。

```
1. 计算差值 d_i = model_A_i - model_B_i
2. 对 |d_i| 进行排名
3. 统计正负秩和
4. 如果分布不同，检验统计量会很小
```

## 检验前的假设检查

| 检验 | 假设 | 如何检查 |
|------|------|--------|
| t 检验 | 正态性 | Shapiro-Wilk 检验或 Q-Q 图 |
| t 检验 | 等方差 | Levene 检验 |
| ANOVA | 正态性 | 每个组进行 Shapiro-Wilk |
| ANOVA | 等方差 | 每个组进行 Levene |
| 独立 t 检验 | 独立性 | 实验设计（不同的训练/测试划分） |

## 多重比较校正

当进行许多检验时，假阳性率会膨胀。

| 校正方法 | 做法 | 强度 | 使用场景 |
|----------|------|------|--------|
| Bonferroni | alpha / k | 非常保守 | 检验数量少（<20）且均重要 |
| Holm-Bonferroni | 逐步递减 | 比邦弗朗尼（Bonferroni）强大 | 邦弗朗尼的一般替代 |
| Benjamini-Hochberg | FDR 控制 | 比 FWER 强大 | 假阳性可接受时（探索性分析） |
| Nemenyi | 所有成对比较 | 用于弗里德曼（Friedman）之后 | 多个模型比较 |

## 常见错误

- 使用 t 检验比较神经网络多次运行的 1 轮结果。这些结果是相关的（成对 t 更合适）。
- 通过选择 sig < 0.05 的最佳结果来 p 值挖掘。始终按多重比较校正。
- 在非常小的测试集上使用 t 检验。效应量过大时考虑贝叶斯方法。
- 错误假设未校正的 5 折 CV 差异是独立的。它们不是（共享训练数据）。使用校正后的 t 检验。
- 混淆统计显著性与实际显著性。p 值不能替代效应量。
- 声称零假设为真因为 p > 0.05。未发现效应不等于没有效应。
- 依赖平均排名而不展示分布。模型排名在不同数据集上可能有很大差异。

## 效应量：仅获得 p 值还不够

为一项发现提供 p 值：

| 检验 | 效应量度量 |
|------|----------|
| t 检验 | Cohen's d: (mean_A - mean_B) / pooled_std |
| ANOVA | eta-squared: SS_between / SS_total |
| 非参数 | Vargha-Delaney A（P(A > B)） |

Cohen's d 指南：小 = 0.2，中 = 0.5，大 = 0.8。

## 最小数据集规模提示

- 配对差异，检验微小效应量（d=0.2）：需要 n > 200
- 配对差异，中效应量（d=0.5）：n > 35
- 配对差异，大效应量（d=0.8）：n > 15
- Wilcoxon 需要比 t 大约多 5% 的数据
- 非参数检验即使对极小样本也有效（n >= 5），但由于离散 ranking 问题，p 值离散
