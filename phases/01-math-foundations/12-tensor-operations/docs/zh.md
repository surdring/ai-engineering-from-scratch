## 概念

### 张量是什么

张量是具有统一数据类型的多维数字数组。维度数称为秩（阶）。每个维度是一个轴。形状是列出每个轴的大小的元组。

```
标量: 秩 0, 形状: ()
向量: 秩 1, 形状: (3,)
矩阵: 秩 2, 形状: (2, 3)
3D 张量: 秩 3, 形状: (2, 2, 2)
4D 张量: 秩 4, 形状: (B, C, H, W)
```

总元素数 = 所有大小的乘积。

### 深度学习中的张量形状

不同数据类型按约定映射到特定张量形状：
- 视觉 (NCHW): (B, C, H, W) = (32, 3, 224, 224)
- NLP: (B, T, D) = (16, 128, 768)
- 注意力: (B, H, T, D) = (16, 12, 128, 64)
- 权重: Linear (out, in)，Conv2D (out_c, in_c, kH, kW)，Embedding (vocab, dim)

PyTorch 使用 NCHW（通道优先），TensorFlow 默认 NHWC（通道后置）。

### 内存布局：步长

内存中的二维数组是一维字节序列。步长告诉你沿每个轴移动一步需要跳过多少元素。

行优先（C 序）：对于形状 (2, 3)，步长为 (3, 1)——前进一行跳 3 个元素。列优先（F 序）：步长为 (1, 2)。转置不移动数据，它交换步长，使张量变为非连续。

### 广播规则

广播让你对不同形状的张量操作而无需复制数据。从右端对齐形状。两个维度在相等或其中一个为 1 时兼容。维度较少的在左侧填充 1。结果取每个维度的最大值。

```
张量 A:     (8, 1, 6, 1)
张量 B:        (7, 1, 5)
填充 B:     (1, 7, 1, 5)
结果:       (8, 7, 6, 5)
```

### Einsum：通用张量操作

爱因斯坦求和用字母标记每个轴。出现在输入但不出现在输出中的轴被求和。出现在两者中的被保留。

关键模式：`i,i->`（点积），`i,j->ij`（外积），`ii->`（迹），`ij->ji`（转置），`bij,bjk->bik`（批量矩阵乘），`bhtd,bhsd->bhts`（注意力分数）。

## 构建它

代码位于 `code/tensors.py`。关键实现：

### 步骤 1：存储和步长

```python
class Tensor:
    def __init__(self, data, shape=None):
        ...

    @staticmethod
    def _compute_strides(shape):
        strides = [1] * len(shape)
        for i in range(len(shape) - 2, -1, -1):
            strides[i] = strides[i + 1] * shape[i + 1]
        return tuple(strides)
```

对于形状 (3, 4)，步长为 (4, 1)。

### 步骤 2：重塑、压缩、扩展

reshape 改变形状而不改变元素顺序。squeeze 移除大小为 1 的轴。unsqueeze 插入一个。unsqueeze 对广播至关重要——偏置向量 (D,) 加到批量 (B, T, D) 需要扩展为 (1, 1, D)。

### 步骤 3：转置和重排

transpose 交换两个轴。permute 重排所有轴。这是 NCHW 和 NHWC 之间的转换方式。转置后张量在内存中非连续——PyTorch 中 view 在非连续张量上会失败。

### 步骤 4：逐元素操作和归约

逐元素操作（加、乘、减）独立应用于每个元素并保持形状。归约（和、均值、最大值）折叠一个或多个轴。CNN 中的全局平均池化：(B, C, H, W).mean(axis=[2,3]) 产生 (B, C)。

## 关键术语

| 术语 | 人们常说的 | 实际含义 |
|------|-----------|---------|
| 步长 | "跳过量" | 沿每个轴移动一步要跳过的元素数 |
| 广播 | "自动拉伸" | 内存高效地操作不同形状的张量 |
| 重塑 | "重新排列" | 改变形状而不改变内存中元素的顺序 |
| 转置 | "交换轴" | 交换两个轴（翻转步长） |
| einsum | "爱因斯坦求和" | 使用字母标记轴的通用张量操作 |
