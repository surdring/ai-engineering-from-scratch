## 问题

一项医学检验准确率为 99%。你检测呈阳性。你实际患病的概率是多少？

大多数人说是 99%。真正的答案取决于疾病有多罕见。如果每 10,000 人中只有 1 人患病，阳性结果只给你约 1% 的患病概率。其他 99% 的阳性结果来自健康人的假警报。

这不是一个陷阱问题。这是贝叶斯定理。每个垃圾邮件过滤器、每个医学诊断、每个量化不确定性的机器学习模型都使用这种精确的推理。你从一个信念开始。你看到证据。你更新。

## 概念

### 从联合概率到贝叶斯

P(A|B) = P(A 且 B) / P(B)，P(B|A) = P(A 且 B) / P(A)。两者有相同的分子 P(A 且 B)：

```
P(A|B) = P(B|A) * P(A) / P(B)
```

这就是贝叶斯定理。四个量，一个等式。

### 四个组成部分

| 部分 | 名称 | 含义 |
|------|------|------|
| P(A\|B) | 后验 | 看到证据 B 后对 A 的更新信念 |
| P(B\|A) | 似然 | 如果 A 为真，证据 B 出现的概率 |
| P(A) | 先验 | 在看到任何证据之前对 A 的信念 |
| P(B) | 证据 | 在所有可能性下看到 B 的总概率 |

证据项 P(B) 作为归一化因子，可以使用全概率公式展开：P(B) = P(B|A)*P(A) + P(B|非A)*P(非A)

### 医学检验示例

疾病影响万分之一的人。检验准确率 99%。

```
P(患病)          = 0.0001     (先验: 疾病罕见)
P(阳性|患病)     = 0.99       (似然: 检验能检测到)
P(阳性|健康)     = 0.01       (假阳性率)

P(患病|阳性) = 0.99 * 0.0001 / 0.010098 = 0.98%
```

不到 1%。先验占主导地位。当一种情况罕见时，即使是准确的检验也大多产生假阳性。

### 垃圾邮件过滤示例

一封含有「彩票」的邮件。是垃圾邮件吗？

```
P(垃圾邮件)              = 0.3
P("彩票"|垃圾邮件)       = 0.05
P("彩票"|非垃圾邮件)     = 0.001

P(垃圾邮件|"彩票") = 0.05 * 0.3 / 0.0157 = 95.5%
```

一个词将概率从 30% 变为 95.5%。

### 朴素贝叶斯：独立性假设

朴素贝叶斯假设所有特征在给定类别的条件下是独立的。给定类别，计算每个特征的概率并将其相乘。忽略分母（对所有类别相同），只比较分子：score(class) = P(class) * P(feature_1|class) * ... * P(feature_n|class)

「朴素」在于独立性假设。在实践中效果出奇地好，因为分类器只需要排序类别。

### 最大似然估计（MLE）

如何从训练数据获得 P(feature|class)？计数。这就是 MLE：选择使观测数据最可能的参数值。

问题：如果一个词在训练期间从未出现在垃圾邮件中，MLE 给它概率为零。一个未见过的词会杀死整个乘积。用拉普拉斯平滑修复：

P(word|class) = (count(word, class) + 1) / (total_words_in_class + vocabulary_size)

### 最大后验估计（MAP）

MLE 问：什么参数最大化 P(data|params)？MAP 问：什么参数最大化 P(params|data)？

P(params|data) ∝ P(data|params) * P(params)

MAP 添加了对参数本身的先验。这在 ML 中等同于 L2 正则化。岭回归中的「岭」惩罚字面上就是权重上的高斯先验。

| 估计方法 | 优化目标 | ML 等价 |
|---------|---------|--------|
| MLE | P(data\|params) | 无正则化训练 |
| MAP | P(data\|params) * P(params) | L2 / L1 正则化 |

### 贝叶斯思维对 ML 为何重要

- **先验就是正则化：** 权重上的高斯先验就是 L2 正则化。拉普拉斯先验就是 L1。
- **后验就是不确定性：** 贝叶斯方法给出分布而非点估计。
- **贝叶斯更新即在线学习：** 今天的后验成为明天的先验。增量更新而非从头重训。
- **模型比较是贝叶斯的：** BIC、边缘似然和贝叶斯因子都使用贝叶斯推理来选择模型。

## 关键术语

| 术语 | 人们常说的 | 实际含义 |
|------|-----------|---------|
| 先验 | "初始信念" | 看到数据前对参数或事件的概率分布 |
| 似然 | "与数据的匹配度" | 给定参数，看到特定数据的概率 |
| 后验 | "更新后的信念" | 看到数据后对参数或事件的概率分布 |
| 朴素贝叶斯 | "简单分类器" | 假设所有特征独立，在文本分类中出奇有效 |
| MLE | "计数" | 选择使数据最可能的参数（频率估计） |
| MAP | "正则化 MLE" | 结合先验（正则化）的 MLE |
