## 问题

你有一个 1000x2000 的矩阵。也许是用户-电影评分。也许是文档-词项频率表。也许是图像的像素值。你需要压缩它、去噪、发现隐藏结构或用它解最小二乘系统。特征分解只适用于方阵，且需要完整的一组线性无关特征向量。

SVD 适用于任何矩阵。任何形状。任何秩。无条件。它把矩阵分解为三个因子，揭示矩阵对空间做了什么。这是整个线性代数中最通用、最有用的因子分解。

## 概念

### SVD 在几何上做了什么

每个矩阵，无论形状如何，都按顺序执行三个操作：旋转、缩放、旋转。SVD 显式地表达了这种分解。

```
A = U * Sigma * V^T

      m x n     m x m    m x n    n x n
     任意     (旋转)    (缩放)   (旋转)
```

- V^T 在输入空间（n 维）中旋转向量
- Sigma 沿每个轴缩放（拉伸或压缩）
- U 将结果旋转到输出空间（m 维）

把 SVD 想象成：给 SVD 一个矩阵。它告诉你：「这个矩阵取一个输入球体，先由 V^T 旋转，然后由 Sigma 拉伸成椭球，再由 U 旋转该椭球。」奇异值就是椭球轴的长度。

### 完整分解

对于形状为 m x n 的矩阵 A：A = U * Sigma * V^T，其中 U 是 m x m 正交矩阵，Sigma 是 m x n 对角矩阵（对角线上为奇异值），V 是 n x n 正交矩阵。奇异值 sigma_1 >= sigma_2 >= ... >= sigma_r > 0，其中 r = rank(A)。

U 的列称为左奇异向量，V 的列称为右奇异向量。Sigma 的对角线项是奇异值，始终非负且通常按降序排列。

### 几何解释

- **右奇异向量（V 的列）：** 输入空间的标准正交基，是矩阵映射为输出空间中正交方向的输入方向。
- **奇异值（Sigma 的对角线）：** 缩放因子。第 i 个奇异值告诉你矩阵沿第 i 个右奇异向量拉伸向量的程度。零奇异值意味着矩阵完全压扁那个方向。
- **左奇异向量（U 的列）：** 输出空间的标准正交基。第 i 个左奇异向量是第 i 个右奇异向量（缩放后）映射到的输出方向。

关系：A * v_i = sigma_i * u_i。这给你了任何矩阵作用的按坐标图形。

### 外积形式

SVD 可写为秩-1 矩阵的和：A = sigma_1*u_1*v_1^T + sigma_2*u_2*v_2^T + ... + sigma_r*u_r*v_r^T

每个项 sigma_i*u_i*v_i^T 是秩-1 矩阵。这是低秩近似的基础。第一项捕获最重要的模式，第二项捕获次重要的，以此类推。截断这个和得到任意秩的最佳可能近似。

### 与特征分解的关系

A^T A 的特征向量是 V（右奇异向量），特征值是 sigma_i^2。因此奇异值始终是实数和非负的。

### 截断 SVD：低秩近似

Eckart-Young-Mirsky 定理指出，A 的最佳秩-k 近似（在 Frobenius 和谱范数下）通过仅保留前 k 个奇异值及其对应向量获得。这不仅是「好」近似，而且可证明是最佳可能的秩-k 近似。

### SVD 用于图像压缩

灰度图像是像素强度矩阵。自然图像具有快速衰减的奇异值。前几个捕获广泛结构（形状、渐变），后面的捕获细节和噪声。在秩 50 处截断通常产生与原始图像几乎相同的图像，同时节省 85% 的存储空间。

### SVD 在推荐系统中的应用

用户-电影评分矩阵被分解为用户因子和电影因子的组合。SVD 找到潜在维度（如「喜剧 vs 严肃」、「动作 vs 剧情」），通过 U 和 V 中的向量表示用户和电影在这些潜在因子上的偏好。这实现了缺失评分的预测。

### 伪逆

SVD 提供计算伪逆的直接途径：A^+ = V * Sigma^+ * U^T，其中 Sigma^+ 对非零奇异值取倒数并对零奇异值置零。伪逆给出超定系统最小范数最小二乘解 x = A^+ b。

## 关键术语

| 术语 | 人们常说的 | 实际含义 |
|------|-----------|---------|
| 奇异值 | "重要性分数" | 矩阵沿对应方向拉伸的倍数 |
| 左奇异向量 | "输出基" | 输出空间到 U 的列的正交基 |
| 右奇异向量 | "输入基" | 输入空间到 V 的列的正交基 |
| 截断 SVD | "低秩近似" | 仅使用前 k 个奇异值——压缩 |
| 伪逆 | "广义逆" | 即使标准逆不存在也能「解」系统 |
