## 问题

模型做出的每个预测都是一个概率分布。每个损失函数衡量预测分布与真实分布的差距。每个训练步骤调整参数以使一个分布看起来更像另一个。没有概率，你无法阅读任何 ML 论文，无法调试任何模型，也无法理解为什么你的训练损失变成了 NaN。

## 概念

### 事件、样本空间与概率

样本空间 S 是所有可能结果的集合。事件是样本空间的子集。概率将事件映射到 0 到 1 之间的数字。

```
抛硬币:
  S = {正面, 反面}
  P(正面) = 0.5,  P(反面) = 0.5

掷一枚骰子:
  S = {1, 2, 3, 4, 5, 6}
  P(偶数) = P({2, 4, 6}) = 3/6 = 0.5
```

三条公理定义了所有概率：
1. 对任意事件 A，P(A) >= 0
2. P(S) = 1（总有事情发生）
3. 当 A 和 B 不能同时发生时，P(A 或 B) = P(A) + P(B)

其他一切（贝叶斯定理、期望、分布）都从这三条规则推导而来。

### 条件概率与独立

P(A|B) 是给定 B 发生时 A 的概率。

```
P(A|B) = P(A 且 B) / P(B)
```

两个事件独立意味着知道一个不会告诉你关于另一个的信息：

```
独立:   P(A|B) = P(A)
等价于: P(A 且 B) = P(A) * P(B)
```

抛硬币是独立的。不放回地抽牌则不是。

### 概率质量函数 vs 概率密度函数

离散随机变量有概率质量函数 (PMF)。每个结果有特定的概率可以直接读取。

连续随机变量有概率密度函数 (PDF)。单点的密度不是概率。概率来自对密度在一个区间上的积分。

这个区别在 ML 中很重要。分类输出是 PMF（离散选择）。VAE 潜空间使用 PDF（连续）。

### 常见分布

**伯努利分布：** 一次试验，两个结果。用于二分类建模。P(X=1)=p, P(X=0)=1-p。

**分类分布：** 一次试验，k 个结果。用于多分类建模（softmax 输出）。

**均匀分布：** 所有结果等可能。用于随机初始化。

**正态分布（高斯分布）：** 钟形曲线。由均值 (mu) 和方差 (sigma^2) 参数化。标准正态分布 68% 数据在 1 个标准差内，95% 在 2 个标准差内，99.7% 在 3 个标准差内。

**泊松分布：** 固定区间内稀有事件的计数。P(X=k) = (lambda^k * e^(-lambda)) / k!。

### 期望值与方差

期望值是加权平均结果。方差衡量围绕均值的散布程度。

在 ML 中，期望值以损失函数的形式出现（数据分布上的平均损失）。方差告诉你模型稳定性。梯度的高方差意味着训练噪声大。

### 联合分布与边缘分布

联合分布 P(X, Y) 共同描述两个随机变量。边缘分布通过对另一个变量求和来获得：

P(X = x) = 对所有 y 求和 P(X = x, Y = y)

### 正态分布为何无处不在

中心极限定理：许多独立随机变量的和（或平均）收敛到正态分布，无论原始分布如何。这就是为什么测量误差近似正态、神经网络权重初始化使用正态分布、SGD 中的梯度噪声近似正态。

### 对数概率

原始概率会导致数值问题。许多小概率相乘会迅速下溢为零。解决方案：使用对数概率——相加而非相乘。这不会改变最大似然解，因为 log 是单调的。log-softmax 使用减去最大 logit 的技巧来防止指数溢出。

## 关键术语

| 术语 | 人们常说的 | 实际含义 |
|------|-----------|---------|
| PMF | "概率表" | 将离散结果映射到概率的函数 |
| PDF | "密度曲线" | 连续随机变量的密度函数，通过积分给出概率 |
| 期望值 | "平均" | 在概率分布上加权平均的结果 |
| 方差 | "散布" | 期望平方偏差，衡量数据离均值有多远 |
| 中心极限定理 | "高斯原因" | 无论原始分布如何，独立随机变量之和趋于正态 |
| 对数概率 | "数值技巧" | 在 log 空间中处理概率以避免下溢，相加而非相乘 |
