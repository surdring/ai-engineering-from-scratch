# 范数与距离

> 你的距离函数定义了「相似」的含义。选错了，下游一切都会出错。

**类型：** 构建
**使用语言：** Python
**前置课程：** 阶段 1，第 01-02 课
**预计时间：** ~90 分钟

## 学习目标

- 从零实现 L1、L2、余弦、马氏、Jaccard 和编辑距离函数
- 为给定的 ML 任务选择合适的距离度量，解释为什么替代方案会失败
- 将 L1 和 L2 范数联系到 LASSO 和 Ridge 正则化及其几何约束区域
- 演示同一数据集在不同度量下产生不同的最近邻

## 问题

两个向量。也许是词嵌入。也许是用户画像。也许是像素数组。你需要知道：它们有多近？

答案完全取决于你选择哪个距离函数。两个数据点可以在一种度量下是最近邻，在另一种度量下相距很远。你的 KNN 分类器、推荐引擎、向量数据库、聚类算法、损失函数——它们都依赖于这个选择。

没有普适的最佳距离。L2 适用于空间数据。余弦相似度在 NLP 中占主导。Jaccard 处理集合。编辑距离处理字符串。马氏距离考虑了相关性。每个都编码了对「相似」含义的不同假设。

## 概念

### L1 范数（曼哈顿距离）

||x||_1 = |x_1| + |x_2| + ... + |x_n|。在城市网格上只能沿轴移动。高维稀疏数据（文本特征、one-hot 编码）和需要抗异常值时使用。L1 正则化（Lasso）通过创建钻石形约束区域将小权重推为零，执行自动特征选择。MAE（平均绝对误差）是预测与目标之间的平均 L1 距离。

### L2 范数（欧氏距离）

||x||_2 = sqrt(x_1^2 + x_2^2 + ... + x_n^2)。n 维的毕达哥拉斯定理。低到中等维的连续数据、可比的特征尺度、物理距离适用。L2 正则化（Ridge）将所有权重向零成比例缩小但很少精确为零——圆形约束区域，轴上没有角落。MSE 对较大误差惩罚更重。

### Lp 范数

||x||_p = (|x_1|^p + ... + |x_n|^p)^(1/p)。不同的 p 产生不同形状的「单位球」：p=1 钻石形，p=2 圆/球，p=inf 方形/超立方体。

### L-无穷范数

||x||_inf = max(|x_1|, ..., |x_n|)。两点间距离由差异最大的单个维度决定。当任何单一维度的最坏偏差很重要时使用——棋盘游戏、制造公差。

### 余弦相似度

cos_sim(a, b) = (a·b) / (||a||_2 * ||b||_2)。衡量角度，忽略大小。从 -1（相反方向）到 +1（相同方向）。余弦距离 = 1 - 余弦相似度。

为什么余弦在 NLP 和嵌入中占主导：文档长度不应影响相似度。一篇关于猫的文章即使比另一篇长两倍，它们的方向仍相同。当大小是噪声而方向是信号时使用。

点积包含了大小信息，这在与归一化嵌入使用时等同于余弦。在检索系统中，大小可能充当隐式的质量或重要性信号。

### 马氏距离

欧氏距离平等对待所有维度。但如果你的特征相关或有不同尺度，L2 会给出误导性结果。马氏距离使用协方差矩阵的逆作为度量张量来考虑数据的协方差结构。它将椭圆数据云转换回球体，然后应用标准欧氏距离。在异常检测和与 PCA 结合使用时特别有用。

### 其他距离

- **Jaccard 距离：** 用于集合——衡量两个集合的重叠。集合操作、推荐系统、文本重复检测。
- **编辑距离（Levenshtein）：** 用于字符串——将一个字符串转换为另一个的最小字符编辑次数。拼写纠正、DNA 序列对齐、模糊匹配。
- **Wasserstein 距离（推土机距离）：** 用于概率分布——将一个分布转换为另一个的最小「功」。GAN 训练（WGAN）、域自适应、直方图比较。

## 距离函数选择指南

| 数据类型 | 推荐度量 | 原因 |
|---------|---------|------|
| 连续数值（低维） | L2 | 标准几何直觉 |
| 高维稀疏数据 | L1 或余弦 | 高维中 L2 失效 |
| 文本/嵌入 | 余弦 | 忽略大小，关注方向 |
| 相关特征 | 马氏 | 考虑协方差 |
| 集合/标签 | Jaccard | 衡量交集/并集重叠 |
| 字符串 | 编辑 | 顺序/对齐敏感 |
| 概率分布 | Wasserstein | 尊重分布几何 |

## 关键术语

| 术语 | 人们常说的 | 实际含义 |
|------|-----------|---------|
| L1 范数 | "曼哈顿距离" | 绝对差值之和 |
| L2 范数 | "欧氏距离" | 平方差之和的平方根 |
| 余弦相似度 | "角度相似度" | 忽略大小的方向对齐度量 |
| 马氏距离 | "相关感知距离" | 考虑变量间相关性的欧氏距离 |
| L1 正则化 | "Lasso" | 通过 L1 范数惩罚进行稀疏特征选择 |
| L2 正则化 | "Ridge" | 通过 L2 范数惩罚将所有系数缩小到零 |
