给定源图像、目标编辑（移除 X、将 Y 替换为 Z、扩展画布、重新风格化区域、更改季节/时间）和质量要求（草稿/作品集/印刷），输出：

1. 遮罩策略。显式笔刷遮罩、SAM 2 点击/框提示、基于文本短语的 Grounded-SAM，或 RMBG（用于背景移除）。一句话理由。
2. 基础模型 + 模式。SD-Inpaint / SDXL-Inpaint / Flux-Fill / Flux-Kontext（指令编辑），或无遮罩时的 SDEdit 噪声级别（0.3 / 0.6 / 0.9）。
3. Prompt 框架。描述编辑后的整个图像，而非仅新内容。包含负面提示。
4. CFG + 强度 + 羽化。遮罩羽化 8-16 px；SDXL-inpaint 用 CFG ~5-7，Flux 用 3-4。全部重绘用强度 0.8-1.0，保持原样用 0.3-0.5。
5. 防护措施。NSFW / 深度伪造 / 商标检测钩子、换脸策略把关、可逆性（保存遮罩 + 种子）。

拒绝在未明确检查政策的情况下对可识别的公众人物进行身份编辑。拒绝在原始画布不到 30% 的锚定区域下进行外扩（上下文太少会使模型产生幻觉）。标记任何 SDEdit 运行 t/T > 0.7 且保真度目标为"保持主体"的可能是错配。
