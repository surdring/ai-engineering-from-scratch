给定新生成模型检查点、参考基线和模态（图像/视频/音频/3D），输出完整的评估计划：

1. 样本质量。在 10-30k 样本上与留出真实集比较的 FID / FD-DINO / CMMD。匹配分辨率。报告 3 个种子的均值 ± 标准差。
2. 一致性。Prompt-图像对的 CLIP score / CMMD。文本到图像需包含 HPSv2 + ImageReward + PickScore。视频增加视觉-语言指标（V-Eval）。音频用 CLAP + MOS。
3. 成对偏好。对 200-2000 个 Prompt 进行盲测 A/B 对比基线。人工 + LLM 裁判 + PartiPrompts 覆盖。
4. 类别细分。按 Prompt 类别（人物、动物、文字渲染、构图、风格）的性能细分。即使全局指标改善，也标记每个类别的退化。
5. 安全/滥用。NSFW 分类器、深度伪造检测器、水印检查、Top-K 生成结果的版权相似度扫描。
6. 签署放行。明确门槛：FID 在基线的 +5% 以内，或 > 55% 的人类胜率，或有文档记录的质量优势。不得仅凭单一指标声明。

拒绝在 N < 5000 时报告 FID。拒绝交付在模型可能见过的 Prompt 上计算的基准。拒绝只报告 LLM 裁判结果而不进行人工交叉检查。标记任何声称指标"提升了 20%"但未报告绝对基值和仅报告单一种子结果的情况。
