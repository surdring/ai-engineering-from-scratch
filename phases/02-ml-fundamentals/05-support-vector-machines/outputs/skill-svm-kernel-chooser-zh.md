# SVM 核函数选择指南

SVM 由两个选择定义：核函数（决定决策边界的形状）和正则化参数（控制间隔宽度和分类错误的权衡）。选对这两项是区分无用模型和强模型的关键。

## 决策清单

1. 数据是否线性可分（或接近线性）？
   - 是：使用线性核。它更快且更可解释。
   - 否：进入第 2 步。

2. 特征与样本的比例如何？
   - 特征 >> 样本（如文本 TF-IDF）：使用线性核。高维数据通常是线性可分的。RBF 增加复杂度却无收益。
   - 样本 >> 特征（如有 10-50 个特征的表格数据）：RBF 核是默认选择。

3. 决策边界是否预期是光滑的？
   - 光滑、连续的边界：RBF 核
   - 多项式形状的边界：多项式核（从 2 阶或 3 阶开始）
   - 领域知识表明存在特定交互项：匹配阶数的多项式核

4. 数据集有多大？
   - 10,000 样本以下：任何核都可以，RBF 是安全默认选择
   - 10,000 到 100,000：线性核或 LinearSVC（原始形式，每轮 O(n)）
   - 100,000 以上：不要使用核 SVM。切换到线性 SVM、梯度提升或神经网络。

5. 是否缩放了特征？
   - SVM 需要特征缩放。拟合前始终标准化（零均值、单位方差）。未缩放的特征会扭曲间隔几何。

## 核函数选择流程图

```
开始
  |
  v
特征 > 1000 或 特征 >> 样本？
  是 --> 线性核（速度优先用 LinearSVC）
  否 --> 数据集 < 10k 样本？
            是 --> 先尝试 RBF（最佳通用核）
            否 --> 线性核（核 SVM 是 O(n^2) 到 O(n^3)）
```

如果 RBF 效果不好，尝试多项式 2-3 阶。如果仍然不行，该问题可能不适合 SVM。

## 调优 C（正则化）

C 控制对错误分类的惩罚力度。它与正则化强度呈反比。

| C 值 | 效果 | 适用场景 |
|------|------|--------|
| 0.001 - 0.01 | 宽间隔，允许大量违规 | 噪声数据，希望泛化 |
| 0.1 - 1.0 | 平衡 | 良好的起始范围 |
| 10 - 1000 | 窄间隔，很少违规 | 干净数据，需要高准确率 |

调优策略：
- 从 C=1.0 开始
- 在对数尺度上搜索：[0.001, 0.01, 0.1, 1, 10, 100, 1000]
- 使用交叉验证选择最佳值
- 如果最佳 C 在你范围的边界，向该方向扩展范围

## 调优 gamma（RBF 核）

Gamma 控制单个训练点的影响范围。它定义了高斯函数的宽度。

| gamma 值 | 效果 | 适用场景 |
|-----------|------|--------|
| 小（0.001） | 每个点影响大面积。光滑、简单的边界 | 欠拟合或特征少 |
| 中等（auto：1/n_features） | sklearn 默认。合理的起点 | 一般用途 |
| 大（10+） | 每个点仅影响邻近点。复杂、波动的边界 | 有过度拟合风险 |

调优策略：
- 从 gamma="scale"（1 / (n_features * X.var())，sklearn 默认）开始
- 在对数尺度上搜索：[0.001, 0.01, 0.1, 1, 10]
- 低 gamma + 高 C 倾向于过拟合
- 高 gamma + 低 C 倾向于欠拟合

## C 和 gamma 联合调优

C 和 gamma 相互影响。始终一起调优而不是独立调优。

推荐方法：
1. 粗粒度网格搜索：C 在 [0.01, 0.1, 1, 10, 100]，gamma 在 [0.001, 0.01, 0.1, 1, 10]（25 种组合）
2. 找到最佳区域
3. 在最佳区域附近精搜（如 C 在 [5, 10, 20, 50]，gamma 在 [0.05, 0.1, 0.2]）
4. 全程使用 5 折交叉验证

## 常见错误

- 在高维稀疏数据上使用 RBF 核（线性更好且快 100 倍）
- 忘记缩放特征（SVM 中最常见的错误）
- 在噪声数据上设置 C 过高（记忆噪声而非学习边界）
- 在 50k 样本以上的数据集上使用核 SVM（训练时间令人望而却步）
- 不联合调优 C 和 gamma（它们互相补偿）
- 默认使用 5 阶以上多项式（极易过拟合，先尝试 2 或 3 阶）

## 快速参考

| 核 | 适用场景 | 关键参数 | 训练复杂度 |
|----|---------|---------|----------|
| Linear | 文本/TF-IDF，特征多，大数据 | 仅 C | 每轮 O(n) |
| RBF | 通用，10k 样本以下 | C, gamma | O(n^2) 到 O(n^3) |
| Polynomial | 已知多项式关系 | C, degree, coef0 | O(n^2) 到 O(n^3) |
| Sigmoid | 很少有用（等价于两层神经网络） | C, gamma, coef0 | O(n^2) 到 O(n^3) |
