# 集成方法选择指南

集成方法组合多个模型以产生比任何单一模型更好的预测。问题始终是：哪种集成，何时使用？

## 决策清单

1. 当前模型的主要问题是什么？
   - 高方差（过拟合）：使用装袋（Bagging）（随机森林）
   - 高偏差（欠拟合）：使用提升（Boosting）（梯度提升、XGBoost）
   - 两者兼有，或希望获得最高准确率：使用堆叠（Stacking）

2. 有多少数据？
   - 1,000 行以下：随机森林（鲁棒，难以错误配置）
   - 1,000 到 100,000：XGBoost 或 LightGBM（表格数据整体最佳）
   - 100,000 以上：LightGBM（最快的梯度提升，能很好地处理大数据）

3. 能投入多少调优时间？
   - 最少：带默认参数的随机森林（几乎总是有效）
   - 中等：XGBoost 配合 learning_rate=0.1，用早停法调优 n_estimators
   - 最多：LightGBM 或 XGBoost 配合贝叶斯超参数搜索

4. 是否需要可解释性？
   - 是：单棵决策树或带特征重要性的小随机森林
   - 部分：带 SHAP 值的梯度提升
   - 否：堆叠或深层集成

5. 数据是否嘈杂且有大量离群值？
   - 是：随机森林（装袋对噪声鲁棒）
   - 否：梯度提升（可以在干净数据上进一步提升准确率）

## 各方法适用场景

**随机森林（装袋）**：你的安全首选。在自举样本上训练许多树并取平均。降低方差而不增加偏差。在中等数据上几乎不会过拟合。调优需求极小：设置 n_estimators=100-500 并保留默认值。

**AdaBoost**：带样本重新加权的序列提升。配合简单基学习器（决策树桩）效果良好。对离群值和噪声标签敏感，因为它为误分类的样本增加权重。在实践中已基本被梯度提升取代。

**梯度提升（Gradient Boosting）**：每棵新树拟合集成模型当前的残差。降低偏差。表格数据上最强大的方法。需要调优：learning_rate、n_estimators、max_depth、min_child_weight、subsample。

**XGBoost**：带正则化、二阶优化和系统级加速的梯度提升。原生处理缺失值。Kaggle 竞赛和生产环境表格数据机器学习的默认选择。

**LightGBM**：带逐叶生长（非逐层生长）的梯度提升。在大数据集上比 XGBoost 快。使用基于直方图的分裂。最适合 50k 行以上的数据集。

**CatBoost**：带原生类别特征处理的梯度提升。无需 one-hot 编码。当有许多类别特征时很好用。

**堆叠（Stacking）**：在多个多样的基模型预测上训练元学习器。当需要绝对最佳准确率且有计算资源余量时使用。始终通过交叉验证生成基模型预测以避免泄漏。

**投票（Voting）**：最简单的集成。硬投票（多数类）或软投票（平均概率）。无需元学习器即可快速组合 2-3 个多样的模型。

## 常见错误

- 不使用早停法就使用梯度提升（如果运行太多轮会过拟合）
- 设置 learning_rate 太高（超过 0.3 通常会导致不稳定）
- 不调优梯度提升的 max_depth（默认无限或很深的树会过拟合）
- 堆叠时所有模型都是同一类型（多样性才是堆叠的意义所在）
- 在噪声数据上使用 AdaBoost（离群值每轮获得越来越高的权重）
- 期望随机森林修复欠拟合（它降低方差而非偏差）

## 各方法的调优优先级

**随机森林：**
1. n_estimators: 100-500（更多很少有坏处，只是更慢）
2. max_depth: None（让树完全生长）或限制在 10-20 以提速
3. max_features: 分类用 "sqrt"，回归用 "log2" 或 n/3

**XGBoost / LightGBM：**
1. learning_rate: 0.01-0.3（越低越好，如果有计算资源训练更多树）
2. n_estimators: 在验证集上使用早停法而不要猜测
3. max_depth: 3-8（从 6 开始）
4. min_child_weight / min_data_in_leaf: 1-20（越高防止过拟合）
5. subsample: 0.7-1.0
6. colsample_bytree: 0.7-1.0
7. reg_alpha（L1）和 reg_lambda（L2）：0-10

## 快速参考

| 方法 | 降低 | 速度 | 调优工作量 | 最适合 |
|------|-----|------|-----------|--------|
| 随机森林 | 方差 | 快 | 低 | 噪声数据，快速基线 |
| AdaBoost | 偏差 | 快 | 低 | 简单基学习器，干净数据 |
| 梯度提升 | 偏差 | 中等 | 高 | 表格数据，竞赛 |
| XGBoost | 两者 | 快 | 高 | 生产环境表格机器学习 |
| LightGBM | 两者 | 最快 | 高 | 大数据集（50k+ 行） |
| CatBoost | 两者 | 中等 | 中等 | 许多类别特征 |
| 堆叠 | 两者 | 慢 | 高 | 最高准确率，多样的模型 |
| 投票 | 方差 | 快 | 无 | 快速组合 2-3 个模型 |
