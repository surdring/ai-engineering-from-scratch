# AGENTS.md

你正在一个使用智能体工作台的仓库中工作。

在行动之前阅读以下内容：

1. `agent_state.json`——上一次会话停止的位置。
2. `task_board.json`——正在进行的任务，接下来是什么。
3. `docs/agent-rules.md`——启动、禁止、完成、不确定性、审批。
4. `docs/reliability-policy.md`——此工作台设计用来吸收的失败模式。
5. `docs/handoff-protocol.md`——会话结束必须产生什么。
6. `docs/reviewer-rubric.md`——完成的工作如何被评判。

验证命令：参见看板上活动任务中的 `acceptance_criteria`。

包版本：1.0.0
