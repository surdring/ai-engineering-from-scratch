# 感知机模式

感知机（Perceptron）计算输入加权和加偏置，然后应用阶跃函数产生二值输出。它是神经网络的基本单元。

```
output = step(w1*x1 + w2*x2 + ... + wn*xn + bias)
```

## 单个感知机足够的情况

- 问题是线性可分的：一条直线（或超平面）可以划分两个类别
- 逻辑门：AND、OR、NOT、NAND
- 简单阈值决策：「分数是否超过 X？」
- 数据聚类为两个不重叠区域的二分类问题

## 需要多层的情况

- 问题不是线性可分的：没有单条直线可以区分类别
- XOR 和奇偶校验问题
- 任何需要「此而非彼」推理的任务（条件组合）
- 现实世界分类：图像、文本、音频——几乎总是非线性的

## 决策清单

1. 绘制或检查你的数据。你能在类别之间画一条直线边界吗？
   - 是：单个感知机有效
   - 否：至少需要两层
2. 问题是否可以分解为更简单线性决策的 AND/OR 组合？
   - 这种分解告诉你最小网络结构
   - XOR = (A OR B) AND (NOT (A AND B)) = 2 层 3 个感知机
3. 对于超过两个类别的问题，每个类别需要一个输出节点

## 训练规则

```
error = expected - predicted
weight_new = weight_old + learning_rate * error * input
bias_new = bias_old + learning_rate * error
```

如果预测正确，不发生变化。如果错误，权重会调整以减少误差。此规则仅适用于单层感知机。多层网络需要反向传播（Backpropagation）。

## 常见错误

- 试图用单个感知机学习非线性模式（永远无法收敛）
- 设置过高的学习率（权重振荡）或过低（训练极慢）
- 忘记偏置项（没有偏置，决策边界必须经过原点）
- 混淆感知机收敛（对线性可分数据保证收敛）和一般神经网络收敛（不保证收敛）
