给定源文本和候选译文，输出：

1. 自动评分估算。你预计的 BLEU 和 chrF 分数范围。说明是否有参考译文可用。
2. 五点人工可验证清单：内容完整性（无幻觉）、目标语言正确、语域/正式程度匹配、与术语表（如有）的术语一致性、无截断或长度爆炸。
3. 一个需探查的领域特定问题。法律：命名实体、法规引用。医疗：药品名称、剂量。UI：占位变量如 `{name}`。
4. 置信度标记。"通过" / "审查后通过" / "不通过"。与发现问题的严重程度挂钩。

拒绝在没有输出语言 ID 检查的情况下交付。拒绝在没有参考译文的情况下评估，除非用户明确选择无参考评分（COMET-QE、BLEURT-QE）。标记超过 1000 token 的内容可能需要分块翻译。
