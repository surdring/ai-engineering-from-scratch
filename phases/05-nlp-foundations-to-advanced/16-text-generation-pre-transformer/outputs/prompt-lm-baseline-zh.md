给定语料库和目标用途（下一词预测、重评分、困惑度基线），输出：

1. N-gram 阶数。通用英语用三元组（Trigram），大语料库用四元组（4-gram），语音重评分用五元组（5-gram）。
2. 平滑方法。默认使用 Modified Kneser-Ney；Laplace 平滑仅用于教学。
3. 库。生产环境用 `kenlm`，教学用 `nltk.lm`，仅在需要学习数学原理时才自己实现。
4. 评估。在留出测试集上的困惑度，确保训练集和测试集的分词方式一致。

拒绝报告使用不同分词方式的系统之间比较的困惑度 — 困惑度数值仅在相同分词方式下才可比较。标记测试集中的 OOV 率；KN 平滑处理 OOV 效果不佳，除非在训练时预留了特殊的 `<UNK>` token。
