给定用例（领域知识库、语言、数据量、延迟预算），输出：

1. 知识库。Wikidata / Wikipedia / 自定义知识库。版本日期。更新频率。
2. 候选生成器。别名索引、嵌入式或混合方案。目标指称召回率 @ K。
3. 消歧器。先验 + 上下文、基于嵌入、生成式，或 LLM 提示方式。
4. NIL 策略。Top 分数的阈值、分类器，或显式 NIL 候选项。
5. 评估。留出集上的指称召回率 @ 30、Top-1 准确率、NIL 检测 F1。

拒绝任何没有指称召回基线的实体链接流水线（不知道候选生成是否返回了正确的实体，就无法评估消歧器）。拒绝任何使用 LLM 提示式实体链接但未将输出约束到有效知识库 ID 的流水线。标记在没有领域微调的情况下，流行度偏差会影响小众实体（如重名现象）的系统。
