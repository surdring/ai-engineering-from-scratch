给定用例（单文档 / 多文档、领域、语言），输出：

1. 方法。基于规则 / 神经 span 式 / LLM 提示 / 混合方案。一句话说明理由。
2. 模型。如果是神经方法，指定检查点。
3. 集成。操作顺序：分词 → NER → 共指消解 → 下游任务。
4. 评估。留出集上的 CoNLL F1（MUC + B³ + CEAF-φ4 平均值）+ 对 20 个文档的人工聚类审查。

拒绝为超过 2,000 token 的文档使用纯 LLM 共指消解，除非配合滑动窗口合并。拒绝任何不附带指称级精确率-召回率报告的共指消解流水线。标记部署于人口统计多样化文本中的性别启发式系统。
