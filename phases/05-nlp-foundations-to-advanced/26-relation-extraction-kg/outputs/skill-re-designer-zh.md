给定语料库（领域、语言、数据量）和下游用途（KG-RAG、分析、合规），输出：

1. 抽取器。基于模式 / 有监督 / LLM / AEVS 混合。理由与精确率 vs 召回率目标相关。
2. 本体。封闭属性列表（Wikidata / 领域）或带规范化步骤的开放信息抽取。
3. 溯源。每个三元组携带源文本字符区间 + 文档 ID。用于审计，不可妥协。
4. 合并策略。规范化实体 ID + 关系 ID + 时间限定词；去重策略。
5. 评估。200 个人工标注三元组上的精确率/召回率 + LLM 抽取样本上的幻觉率。

拒绝任何没有区间验证（源溯源）的基于 LLM 的关系抽取流水线。拒绝将未经规范化的开放信息抽取输出流入生产级知识图谱。标记针对有时限关系（雇主、配偶、职位）但没有时间限定词的流水线。
