给定用例（RAG / 智能体 / 生成式任务），输出：

1. 指标。忠实性 / 相关性 / 上下文精确率 / 上下文召回率 + 任何带标准的自定义 G-Eval 指标。
2. 评判模型。指定模型 + 版本，成本 vs 准确率的理由。
3. 校准。人工标注集规模，目标 Spearman rho 与人类相关性 > 0.7。
4. 数据集版本管理。标签策略、变更日志、分层。
5. CI 门控。各指标阈值、回归窗口逻辑、底部百分位告警。

拒绝依赖未在 ≥ 50 个人工标注样本上测试的评判器。拒绝自评估（同一模型既生成又评判）。拒绝仅报告汇总指标而不展示底部 10%。标记评判模型升级时没有进行平行基线评估的流水线。
