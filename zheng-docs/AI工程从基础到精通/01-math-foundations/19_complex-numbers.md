# AI 中的复数

> -1 的平方根并非虚幻。它是旋转、频率和信号处理半壁江山的关键。

**类型：** 学习
**使用语言：** Python
**前置课程：** 阶段 1，第 01-04 课
**预计时间：** ~60 分钟

## 学习目标

- 在直角坐标和极坐标形式下进行复数运算（加、乘、除、共轭）
- 应用欧拉公式在复指数和三角函数之间转换
- 使用单位复根实现离散傅里叶变换
- 解释复数旋转如何构成 RoPE 和 Transformer 中正弦位置编码的基础

## 问题

你打开一篇傅里叶变换的论文，到处都是 i。你看 Transformer 位置编码，看到不同频率的 sin 和 cos——复指数的实部和虚部。你读量子计算，发现一切都在复向量空间中表达。

复数看似抽象。一个建立在 -1 平方根上的数系感觉像数学把戏。但它不是把戏。它是旋转和振荡的自然语言。每当有什么东西旋转、振动或振荡时，复数就是正确的工具。

不理解复数，你就无法理解离散傅里叶变换（DFT）、FFT、现代语言模型中 RoPE（Rotary Position Embedding）的工作原理，以及原始 Transformer 论文中正弦位置编码为什么使用那些频率。

## 概念

### 什么是复数？

z = a + bi，其中 a 是实部，b 是虚部，i 是虚数单位（i^2 = -1）。你将数轴扩展到平面。每个复数都是这个平面中的一个点。

### 复数运算

**加法：** (a+bi)+(c+di) = (a+c)+(b+d)i。**乘法：** (a+bi)(c+di) = (ac-bd)+(ad+bc)i。**共轭：** a+bi 的共轭是 a-bi。**除法：** 分子分母同乘分母的共轭。

### 复平面

水平轴是实轴，垂直轴是虚轴。z = 3+2i 对应点 (3,2)。一个复数同时是一个点和一个从原点出发的向量。这种双重解释使复数对几何有用。

### 极坐标形式

z = r*(cos(theta) + i*sin(theta))，其中 r = |z|（模长），theta 是相角。直角坐标适合加法，极坐标适合乘法。极坐标形式下的乘法：幅值相乘，角度相加。

### 欧拉公式

e^(i*theta) = cos(theta) + i*sin(theta)。这是本课最重要的公式。当 theta=π 时：e^(iπ) + 1 = 0——五个基本常数（e, i, π, 1, 0）通过一个方程连接。

### 为什么欧拉公式对 ML 重要

e^(i*theta) 随着 theta 变化在单位圆上描迹。这意味着复指数就是旋转，而旋转在信号处理和 ML 中无处不在。

### 与二维旋转的联系

将复数 (x+yi) 乘以 e^(i*theta) 将点 (x,y) 绕原点旋转角度 theta。复数乘法就是二维旋转。旋转矩阵只是用矩阵记法写的复数乘法。

### 旋转相量

复指数 e^(i*omega*t) 是一个以角频率 omega 绕单位圆旋转的点。其实部是 cos(omega*t)，虚部是 sin(omega*t)。正弦信号是旋转复数的影子。这使相移成为角度偏移，幅度变化成为幅值变化。

### 单位根

N 次单位根：w_k = e^(2π*i*k/N)，k=0,1,...,N-1。在单位圆上均匀分布的 N 个点。它们是 DFT 的基础，DFT 将信号分解为这 N 个等间距频率上的分量。

### RoPE：旋转位置嵌入

传统的 Transformer 位置编码只是给嵌入加上不同频率的正弦值。RoPE 通过将嵌入旋转一个与位置成比例的角度来编码位置信息——位置编码就写在复数乘法中。

## 关键术语

| 术语 | 人们常说的 | 实际含义 |
|------|-----------|---------|
| 虚数单位 | "i" | 满足 i^2 = -1 的数 |
| 共轭 | "翻转符号" | 改变虚部符号：a+bi → a-bi |
| 极坐标形式 | "幅值+角度" | 用距离和角度而非 x,y 描述复数 |
| 欧拉公式 | "桥梁" | e^(iθ) = cos(θ) + i*sin(θ) |
| 单位根 | "DFT 基础" | 将平面等分为 N 个部分以分解频率 |
