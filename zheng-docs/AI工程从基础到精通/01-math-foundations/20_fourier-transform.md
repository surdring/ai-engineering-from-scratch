# 傅里叶变换

> 每个信号都是正弦波之和。傅里叶变换告诉你哪些正弦波。

**类型：** 构建
**使用语言：** Python
**前置课程：** 阶段 1，第 01-04、19 课
**预计时间：** ~90 分钟

## 学习目标

- 从零实现 DFT，并对 O(N log N) 的 Cooley-Tukey FFT 进行验证
- 解释频率系数：从信号中提取幅度、相位和功率谱
- 应用卷积定理通过 FFT 乘法执行卷积
- 将傅里叶频率分解联系到 Transformer 位置编码和 CNN 卷积层

## 问题

音频录音是随时间变化的压力测量序列。股价是按天的数值序列。图像是空间上的像素强度网格。这些都是时域（或空间域）中的数据。

但许多模式在时域中是不可见的。这段音频是纯音还是和弦？这个股价有按周的周期吗？这个图像有重复的纹理吗？这些问题关乎频率内容，而时域隐藏了它。

傅里叶变换将数据从时域转换到频域。它将信号分解为不同频率的正弦波，每个正弦波有一个幅度（多强）和一个相位（从哪里开始）。

这对 ML 很重要：CNN 执行卷积，而卷积就是频域中的乘法；Transformer 位置编码使用频率分解来表示位置；音频模型在频谱图（声音的频率表示）上操作；时间序列模型寻找周期性模式。

## 概念

### DFT 定义

给定 N 个样本 x[0],...,x[N-1]，DFT 产生 N 个频率系数 X[0],...,X[N-1]：

```
X[k] = sum_{n=0}^{N-1} x[n] * e^(-2π*i*k*n/N)
```

每个 X[k] 是复数。|X[k]| 是幅度，angle(X[k]) 是相位。关键洞察：e^(-2π*i*k*n/N) 是频率为 k 的旋转相量。DFT 计算信号与每个等间距频率之间的相关性。

### 各系数的含义

- **X[0]：DC 分量。** 所有样本之和，与均值成比例。代表常数（零频率）偏移。
- **X[k]（1<=k<=N/2）：正频率。** X[k] 代表每 N 个样本 k 个周期。k 越高频率越高。
- **X[N/2]：奈奎斯特频率。** N 个样本能表示的最高频率。超过此频率就会出现混叠。
- **X[k]（N/2<k<N）：负频率。** 对实信号，X[N-k] = conj(X[k])。有用信息在前 N/2+1 个系数中。

### 逆 DFT

x[n] = (1/N) * sum_{k=0}^{N-1} X[k] * e^(2π*i*k*n/N)。完全重构。无信息丢失。DFT 是基变换——用不同坐标系重新表达相同信息。

### FFT：使其快速

DFT 是 O(N^2)。FFT 在 O(N log N) 中计算相同结果。Cooley-Tukey 算法通过分治法工作：将信号分成偶数索引和奇数索引样本，递归计算各一半的 DFT，使用旋转因子组合。

```
X[k] = E[k] + e^(-2π*i*k/N) * O[k]
X[k+N/2] = E[k] - e^(-2π*i*k/N) * O[k]
```

每层递归做 O(N) 工作，有 log2(N) 层。总计 O(N log N)。FFT 要求信号长度是 2 的幂。

### 频谱分析

功率谱 P[k] = |X[k]|^2 显示每个频率有多少能量。相位谱 phi[k] = atan2(X[k].imag, X[k].real)。大多数分析任务关心功率谱而忽略相位。

### 频率分辨率

频率分辨率 delta_f = fs/N。要区分两个接近的频率需要更多样本。捕获高频需要更高采样率。奈奎斯特频率 = fs/2。

### 卷积定理

信号处理中最重要的结果之一，与 CNN 直接相关。**时域中的卷积等于频域中的逐点乘法。** x * h = IFFT(FFT(x) · FFT(h))。注意 DFT 计算的是循环卷积，对于线性卷积需零填充。

### 加窗

DFT 假设信号是周期的。如果信号起止不同值，边界不连续会表现为虚假的高频内容（频谱泄漏）。加窗通过将信号两端逐渐减小为零来减少泄漏。常用窗口：矩形（平坦，主瓣最窄但旁瓣最高）、Hann（升余弦，通用）、Hamming（改进余弦，音频处理）、Blackman（三重余弦，需要极高旁瓣抑制时）。

## 关键术语

| 术语 | 人们常说的 | 实际含义 |
|------|-----------|---------|
| DFT | "傅里叶变换" | 将信号分解为 N 个等间距频率分量 |
| FFT | "快速傅里叶变换" | O(N log N) 的 DFT 分治实现 |
| 卷积定理 | "频域乘法" | 时域卷积 = 频域相乘 |
| 频谱图 | "频域图" | 随时间变化的频率内容的视觉表示 |
| 奈奎斯特频率 | "一半采样率" | 被你采样率限制的可表示最高频率 |
